PA 4
1. 3D 모델로 장면 구성하기 [3점]
1.1 OBJ 파일에서 텍스처가 포함된 서로 다른 10개 이상의 3D 모델을 로드할 것
1.2 선택한 모델 중 하나를 10개 이상 복제하고, 인스턴싱 기법을 사용하여 서로 다른 위치에 배치할 것
1.3 3개 이상의 객체를 장면 내에서 계속해서 이동 및 회전시킬 것

2. 카메라 이동 및 사용자 입력을 통한 장면 탐색 [2점]
2.1 방향키 (위/아래/왼쪽/오른쪽) 를 이용해 카메라 이동 제어 (전진/후진/좌/우 이동)
2.2 마우스 움직임으로 카메라 시야 방향 제어

※ 장면 전체를 자유롭게 탐색하며 모든 객체를 관찰할 수 있어야 함

3. 2D 이미지 출력하기 [3점]
3.1 타이틀 화면 추가: 장면 제목, 제작자 이름, 조작 방법 표시
3.2 배경 이미지 및 빌보드 이미지 출력
3.3 렌더링 성능 정보 표시:

FPS (초당 프레임 수)
CPU 사용률
객체 수
폴리곤 수
화면 해상도

4. 계획 발표 및 결과 시연 [2점]
4.1 계획 발표 자료 제출 (PPT 또는 PDF):
타이틀
메인 장면 (스케치나 참고 이미지 포함)
사용한 리소스

※ 실제 구현 결과는 계획과 유사해야 함

4.2 최종 보고서 제출 (PDF, 1페이지 내외):
조작 방법
구현 기능
구현 시 이슈 및 난이도 등 기술

※ 구현 조건
프로그램은 x86 (32비트) 릴리즈 모드에서 실행 가능해야 함

프레임워크 클래스는 수정은 가능하지만 삭제는 금지

모든 구현 기능은 동일한 프로그램 내에서 실행되어야 함
